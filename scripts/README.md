# Ad-hoc Scripts

Utility scripts for debugging, testing, and one-off tasks. Not part of the main package.

## Scripts

| Script | Purpose |
|--------|---------|
| `check_lamplight_dates.py` | Find emails from lamplightdb.co.uk domain |
| `debug_effi_core.py` | Test effi-core MCP connection |
| `get_sent_email_dec9.py` | Get sent emails with domain/subject filter |
| `outlook_recent_emails.py` | Interactive domain categorization tool |
| `read_domain_emails.py` | Read all emails from a specific domain |
| `read_lamplight_full.py` | Full Lamplight email reader |
| `sent_to_domain.py` | Read emails sent to a specific domain |
| `test_domains.py` | Count emails in DMSforLegal matters |
